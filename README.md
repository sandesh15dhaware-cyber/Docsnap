# DocSnap

Offline-capable document scanner PWA. Capture or upload images, apply filters, reorder pages, export as PDF. All scans saved to device via IndexedDB.

## Features
- Camera capture & image upload
- Filters: B&W, Document, Sharp
- Brightness & contrast controls
- Rotate, delete, reorder pages
- Export as A4 PDF
- Offline support (PWA)
- Local device storage (IndexedDB)

## Install on Phone
1. Open the deployed URL in Chrome
2. Tap ⋮ menu → "Add to Home screen"
3. App icon appears — works offline

## Development
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub → auto-deploys on Vercel.
